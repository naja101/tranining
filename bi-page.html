✓ เรียนรู้การเตรียมข้อมูล, สร้าง Data Model, เขียน DAX และออกแบบ Dashboard สำหรับผู้บริหาร

สิ่งที่จะได้เรียน (Course Outline)

เนื้อหาครบตั้งแต่พื้นฐานจนสร้าง Dashboard สำหรับงานจริงในองค์กร

Module 1 : พื้นฐาน Power BI สำหรับคนทำงานธุรกิจ

ทำความเข้าใจภาพรวมของ Power BI และการนำไปใช้ในองค์กร

✨ รู้จัก Power BI Desktop / Service / Mobile

✨ เข้าใจโครงสร้าง Report, Model, Dashboard

✨ เตรียมเครื่องมือและไฟล์ตัวอย่าง

Module 2 : เตรียมและจัดการข้อมูล (Power Query)

ดึงข้อมูลจากหลายแหล่งและจัดรูปแบบให้พร้อมวิเคราะห์

✨ นำเข้าข้อมูลจาก Excel / CSV / Database

✨ ลบข้อมูลซ้ำ แยก/รวมคอลัมน์ แก้ประเภทข้อมูล

✨ สร้าง Date Table สำหรับวิเคราะห์ตามเวลา

Module 3 : Data Modeling for Business

ออกแบบ Data Model ที่ดีสำหรับการวิเคราะห์เชิงธุรกิจ

✨ แนวคิด Star Schema แบบเข้าใจง่าย

✨ กำหนด Relationships ระหว่างตาราง

✨ Best Practice ในการออกแบบ Model

Module 4 : สร้าง KPI ด้วย DAX

สร้าง Measure เพื่อวัดผลและตอบคำถามทางธุรกิจ

✨ SUM, COUNTROWS, AVERAGE, DIVIDE

✨ CALCULATE, FILTER เบื้องต้น

✨ ยอดขาย, กำไร, % Growth, เทียบเป้าหมาย

Module 5 : ออกแบบ Report & Dashboard ระดับผู้บริหาร

ออกแบบหน้ารายงานให้ทั้งสวยและใช้งานง่าย

✨ หลัก Data Visualization สำหรับคนธุรกิจ

✨ ใช้ Slicer, Filter, Drill-through, Tooltip

✨ สร้าง Executive Dashboard 1 หน้า

Module 6 - 7 : ช้งานจริงในองค์กร + Business Case Workshop

ปิดท้ายด้วยโปรเจกต์จากเคสธุรกิจจริง

✨ Publish & Share Report บน Power BI Service

✨ ตั้งค่า Scheduled Refresh ให้ข้อมูลอัปเดตอัตโนมัติ

✨ ทำโปรเจกต์จริง 1 ชิ้น: Sales / Finance / HR Dashboard

ทักษะที่ได้หลังเรียนจบ (Learning Outcomes)

คุณจะได้ทั้งความเข้าใจ แนวคิด และทักษะเชิงปฏิบัติที่ใช้ได้จริงในงาน

เข้าใจ Power BI ในบริบทธุรกิจ

มองเห็นภาพว่าควรใช้ Power BI เมื่อไร และช่วยองค์กรยังไง

เตรียมข้อมูลและสร้าง Data Model ที่ดี

ออกแบบโครงสร้างข้อมูลที่รองรับการวิเคราะห์ระยะยาว

เขียน DAX เพื่อสร้าง KPI สำคัญของธุรกิจ

สร้างตัววัดยอดขาย กำไร การเติบโต และตัวชี้วัดอื่น ๆ

ออกแบบ Dashboard ที่ผู้บริหารใช้ได้จริง

สร้างรายงานที่อ่านง่าย ตอบคำถามทางธุรกิจได้ในไม่กี่คลิก

ใช้งาน Power BI ภายในองค์กร

Publish, Share และตั้งค่า Refresh เพื่อให้รายงานอัปเดตอัตโนมัติ

มี Portfolio Dashboard จริง

ใช้เป็นผลงานยื่นสมัครงาน หรือเสนอให้หัวหน้า / ผู้บริหาร

เหมาะสำหรับใคร

คอร์สนี้ออกแบบมาโดยเฉพาะสำหรับ 3 กลุ่มต่อไปนี้


1. ผู้ที่กำลังจะหางาน / เปลี่ยนสายงาน

ต้องการเพิ่มสกิลด้าน Data & Dashboard ให้เรซูเม่น่าสนใจ มีผลงาน Power BI Dashboard จริงใช้เป็น Portfolio ในการสมัครงานสาย Data / Business Analyst / Reporting

2. พนักงานที่ต้องการเติบโตในสายงาน

ใช้ Excel เป็นอยู่แล้ว แต่อยากยกระดับไปสู่การวิเคราะห์ข้อมูลและทำ Dashboard เพื่อลดงานรายงานซ้ำ ๆ และมีเวลาโฟกัสงานเชิงกลยุทธ์มากขึ้น

3. HR / ผู้จัดการฝ่ายอบรม (L&D) ในองค์กร

มองหาคอร์สที่ช่วย Upskill พนักงานด้านการอ่านและวิเคราะห์ข้อมูล เปลี่ยนรายงานแบบเดิม ๆ ให้กลายเป็น Dashboard Interactive ที่ผู้บริหารใช้ตัดสินใจได้จริง
